# SimCity 2013 Mods Mirror

This repository provides a temporary backup of SimCity 2013 mods while Simtropolis is down.

## ⚠️ Important Notes
- This repo only contains the mod files.
- Installation instructions and detailed descriptions are **not included**.
- For proper usage, please check Simtropolis once it is available again.

## Purpose
- Ensure mods remain accessible to the community.
- Provide a temporary download source.

## Disclaimer
These mods are community-created. Use at your own risk.  
For official documentation and support, visit [Simtropolis](https://www.simtropolis.com) when it is back online.
